# iswo-academy

A project powered by Next.js 14 and shadcn/ui for build an awesome web page where everyone can share their knowledge as videos etcetera.

## Usage

To run de project locally just have to install and run make sure you have node 20+

```bash
npm install
npm run dev
```

## Features

- Next.js 14
- Radix UI Primitives
- Tailwind CSS
- Icons from [TablerIcons](https://lucide.dev)
- Dark mode with `next-themes`
- Tailwind CSS class sorting, merging and linting.
